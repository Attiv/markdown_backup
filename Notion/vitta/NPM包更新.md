使用第三方:

```Plain
npm install -g npm-check-updates

ncu // 查看可更新包　　　　　

ncu -u // 更新package.json　　　　　

npm install // 升级到最新版本
```

更新全局包:

```Plain
要更新全局包的话，那就再全局安装一下：
npm install -g <package>：
```
